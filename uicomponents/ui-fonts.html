Custom Fonts

You can set change the fonts using these Bootstrap variables:

  • ?font-family-base=Georgia,Times,Times New Roman,serif
  • ?headings-font-family=Century Gothic,sans-serif
  • Reset fonts

If you add &google-fonts=y to the URL, Google Fonts are automatically imported.

  • ?google-fonts=y&font-family-base=Roboto,Open Sans,sans-serif
  • ?google-fonts=y&headings-font-family=Work Sans,Calibri,sans-serif
  • Reset fonts

To use your own fonts, import them before bootstraptheme.css, like this:

{% apply only_source %} {% end %}
