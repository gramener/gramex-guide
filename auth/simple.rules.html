This AuthHandler enforces the following User Attribute Modification Rules:

  • If the email attribute ends with "@gmail.com", then the role attribute is set to "guest".
  • If the role attribute is "admin", then the location attribute is set to "NYC".

Therefore,

  • If you log in as beta, your role is "guest".
  • If you log in as alpha, your location will be "NYC".
